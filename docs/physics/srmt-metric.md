# SRMT affine-fit quality `q` — derivation and alternatives

## Statement of the conjecture under test

The Superspace-Relational Modular Time (SRMT) conjecture in this project's
Wheeler–DeWitt setting asserts: for a bipartition of the minisuperspace
wavefunction `χ(a, φ₁, φ₂)` along the DeWitt-timelike scale-factor axis
`a`, the **modular Hamiltonian** `K_A = −log ρ_A` of the reduced density
matrix on one side of the cut is unitarily equivalent, on the low-lying
spectrum, to the **Hamilton–Jacobi (HJ) operator** generated by
`U(a, φ) = a · V(φ) − a`, evaluated on the slice orthogonal to the cut.

That is, let `{s_n}` be the Schmidt singular values across the cut and

```text
K_n := −log(s_n² + ε),        n = 0, 1, …, r−1
```

be the modular spectrum (ascending), and let `{E_n}` be the leading `r`
eigenvalues of the discrete HJ operator on the slice. The strict SRMT
conjecture is

```text
(C1)     K_n = E_n + c,     c constant in n,
```

for clock `a` but **not** for clocks `φ₁, φ₂`. `c` absorbs the
zero-of-energy convention (HJ eigenvalues are defined up to an additive
constant; modular `K` is defined up to the trace normalisation `tr ρ = 1`,
which fixes the arithmetic mean of `K_n` but not its absolute zero).

## The affine-fit metric `q_affine`

The live diagnostic (`src/lib/physics/srmt/affineFit.ts`) reports

```text
q_affine = Σ_n (K_n − (α·E_n + β))² / Σ_n K_n²                      (1)
```

where `(α, β)` are the linear-least-squares fit minimisers across the
first `rankCap` modes. The residual is dimensionless and scale-invariant
in `E` (α absorbs scale). It is **not** generally shift-invariant in `K`:
while β removes shifts from the numerator, the denominator `Σ K_n²` is
not shift-invariant, so translating `K` changes `q` when the fit is
imperfect.

`q_affine = 0` iff `K_n = α·E_n + β` for some `(α, β)`. This tests a
**strictly weaker** claim than (C1):

```text
(C2)     K_n = α·E_n + β,     α > 0, β ∈ ℝ.
```

Under (C2) the two spectra are affinely related — the same linear order
— but not numerically identical. A universe where (C1) fails but (C2)
holds is one where clock `a` is an affine image of HJ: the same physics,
up to a rescaling of time. Whether that counts as "SRMT holds" depends on
how strictly one reads the conjecture. **This project's position: `q_affine`
alone is not sufficient evidence for (C1).**

### Why α-freedom matters

The HJ operator is built on a discretised PDE whose leading eigenvalues
scale with `1/Δx²`. The modular `K_n` are log-singular values of a
finite-dimensional Schmidt decomposition — no PDE discretisation appears.
Equating `K` and `E` *numerically* requires matching units, which in turn
requires a choice of `ℏ` scaling, grid spacing convention, and a factor
that converts between canonical pair variables. Absorbing this conversion
into an unconstrained `α` is exactly what the affine fit does.

The price: `q_affine` can reward a theory where the two spectra have the
same *shape* but disagree on *absolute magnitude* by an order of
magnitude — equivalent, numerically, to "clock `a` runs at a different
rate than HJ but is otherwise HJ". That is not the conjecture.

## The rigid-fit metric `q_rigid`

Fix α = 1 and minimise over β only:

```text
β* = mean_n K_n − mean_n E_n                                        (2)
q_rigid = Σ_n (K_n − E_n − β*)² / Σ_n K_n²                          (3)
```

This is the test of (C1) directly: β absorbs the zero-of-energy, α is
**not** free. By construction

```text
q_rigid ≥ q_affine,                                                  (4)
```

with equality iff the affine fit already chose α = 1. The gap
`q_rigid − q_affine` is a direct measure of how much the affine fit was
exploiting α to lower the residual — i.e. how much scale disagreement
between `K` and `E` the affine fit is hiding.

### Recommended reporting

Both metrics are computed per sweep point and emitted in the CSV export.
The plot still shows `q_affine` because it has the cleaner interpretation
("same spectral order") and is the number Sec. 5 of the thesis will cite.
But the CSV carries `q_rigid` alongside, and any numerical claim that
relies on the modular–HJ match MUST be checked against both:

- `q_rigid ≪ 1` is strong evidence for (C1).
- `q_affine ≪ 1 ∧ q_rigid ≳ 1` is evidence for (C2) only — the weaker
  claim.
- `q_rigid / q_affine > 10` flags a large α-dependence, which in turn
  flags a unit / scale mismatch that must be understood before reading
  the result as "SRMT holds".

## Jackknife σ

The leave-one-out jackknife (`jackknifeAffineFitStdev`) is computed for
**both** metrics (see `jackknifeRigidFitStdev`). No
published `q` lacks its σ.
