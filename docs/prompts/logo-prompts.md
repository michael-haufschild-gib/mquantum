# MDimension Logo Prompts

Generated for Nano Banana / Gemini Image Generation.

## Requirements

- Square format, centered composition
- No text
- Must scale from 16px favicon to 1024px+ splash screen
- Dark background matching app aesthetic
- Neon accents (cyan, blue, magenta)
